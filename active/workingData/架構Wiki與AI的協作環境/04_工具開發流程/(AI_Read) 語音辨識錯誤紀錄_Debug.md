## 分類一：快捷鍵類誤辨

> 共同規律：語音辨識對快捷鍵組合辨識能力差，「Ctrl」容易辨識正確，後面的字母常被誤解為音近中文字。

| 原始辨識 | 正確內容 | 說明 | 來源影片 |
|---------|---------|------|---------|
| Ctrl+勾 | Ctrl+J | 複製圖層，「勾」= J 的口語說法 | V7、V8（多支影片共通） |
| 三件加 t | Ctrl+Shift+Alt+T | 重複變換，「三件」= 三個修飾鍵 | V5 |
| 三件家意 | Shift+Ctrl+Alt+E | 蓋印圖層，「家意」≈ E 的發音 | V4 |
| Ctrl+2 反选 | Ctrl+I（舊版）/ Ctrl+Shift+I（現版本）= 反轉選取 | 講者口音使「I」音近「2」；現版 PS（CC 2024+）為 Ctrl+Shift+I | V7 |

## 分類二：軟體術語音近誤辨

> 共同規律：PS / AI 的專業術語，中文發音與其他詞語高度相近，是誤辨重災區。

| 原始辨識 | 正確術語（繁體） | 英文對應 | 說明 | 來源影片 |
|---------|---------|---------|------|---------|
| 套缩工具 | 套索工具 | Lasso Tool | 「縮」≈「索」，音近 | V7 |
| 鞋面浮雕 | 斜面和浮雕 | Bevel and Emboss | 嚴重音近誤辨，全段多次出現 | V8 |
| 预知 / 预知色阶 | 閾值 | Threshold | yù zhī ≈ yù zhí，音近 | V3 |
| 雨化 / 雨画 | 羽化 | Feather | 音近誤辨，多支影片共通 | V1、V7 |
| 纯色涂层 | 純色填充圖層 | Solid Color fill layer | 「涂层」≈「圖層」，但指特定填充圖層類型 | V7 |
| 涂层 | 圖層 | Layer | 音近誤辨（全段通用） | V7、V8 |
| 色劑 | 色階 | Levels | 「劑」≈「階」，音近 | V4 |
| 綠鏡 | 濾鏡 | Filter | 「綠」≈「濾」，聲調誤辨 | V4 |
| 綠鏡庫 | 濾鏡庫 | Filter Gallery | 同上 | V4 |
| 举行 | 矩形 | Rectangle | 「举」≈「矩」，音近 | V5 |
| 素质 | 複製 | Duplicate | 音近誤辨 | V1 |
| 简单工具 | 減淡工具 | Dodge Tool | 「简单」≈「减淡」，[辨識存疑，V3 Phase 4 待實機確認] | V3 |
| 只能對象 / 整人只能對象 | 智慧型物件 | Smart Object | 嚴重音近誤辨 | V4 |
| 画品 | 畫筆 | Brush | 「品」≈「筆」，音近 | V4 |

## 分類三：操作描述音近誤辨

> 共同規律：口頭說明操作時，句子中的動詞或量詞發音相近，導致整句意思偏移。

| 原始辨識 | 正確意思 | 說明 | 來源影片 |
|---------|---------|------|---------|
| 给一丢丢雨画 | 給一點羽化（Feather） | 「雨画」= 羽化，「丢丢」= 一點點（量詞誤辨） | V1 |
| 这里的竖直拉大 | Distance（距離）數值拉大 | 動感模糊參數，「竖直」為方向描述，被誤接成量詞 | V1 |
| 跟句話面調整 | 根據畫面調整 | 「跟句」= 根據，「話面」= 畫面 | V4 |
| 深度打小 | 深度、大小（兩個獨立參數） | 語音連讀誤辨為一個詞，實為兩個獨立參數名稱 | V8 |
| 移個一個 | 移動滑桿調整模糊數值 | 操作描述音近誤辨 | V4 |
| 停架關鍵點 | 在文字轉折點添加關鍵點（blur pin） | 「停架」≈「添加」 | V4 |
| 模糊化狼長就模糊 | 模糊收藏館 → 景色模糊（Field Blur） | 路徑描述嚴重誤辨 | V4 |

## 分類五：兩岸術語差異

> 非辨識錯誤，而是影片用簡體 PS 術語，繁體 PS 名稱不同。需換算後寫入規格草稿。

| 影片術語（簡體） | 繁體 PS 對應名稱 | 說明 |
|---------|---------|------|
| 光澤等高線 | 光澤輪廓 | 同一功能，繁簡 PS 名稱不同 |
| 内阴影 距離 | 間距 | 繁體 PS 用「間距」 |
| 阻塞 | 填塞 | 繁體 PS 用「填塞」 |
| 等高线（斜面浮雕子項） | 輪廓 | 繁體 PS 中斜面和浮雕下方的子選項稱「輪廓」 |
