## 效果描述

在 Illustrator 中對文字執行「鋸齒化」效果，透過不同描邊粗細與鋸齒化程度的三份文字疊加，產生手繪凌亂線條感的藝術字體效果。

**適用情境**：海報標題、藝術字設計、手繪風格視覺

> ⚠️ **字型相依性**：此效果與初始字型高度相關。許多字型套用後效果不明顯或完全無效（如過細的無襯線體）；建議優先使用已驗證的字型（見末節），或在步驟開始前先確認字型。

## 操作步驟（結構化，v0.3 確認版）

> 軟體：Adobe Illustrator

```
步驟 0：⏸ 確認字型
         告知使用者：此效果對字型有強烈相依性，建議選用已驗證字型（見末節「已驗證字型」）
         若有已驗證字型 → 套用後繼續
         若使用自訂字型 → 記下字型名稱，步驟 8 確認效果時一併評估是否可用

步驟 1：打好文字 → 右鍵「建立外框」（Create Outlines）
原文：你就打上文字 然后右键创建轮廓

步驟 2：切換填塞與筆畫（Shift+X），讓文字以描邊顯示
（原草稿誤記為「改為描邊模式」，實際快捷鍵為 Shift+X）
原文：再点击左下角的箭头改成描编

步驟 3：複製四次（共五份，含原本）→ 全選 → 水平居中 + 垂直居中
（講者口誤說「複製五個」，包含原本只需複製四次）
原文：接着再复制五个出来 全选给它剧中对齐

步驟 4：全選五個物件 → 移動（水平：0、垂直：45mm）→ 選擇「拷貝」→ 執行兩次
（此步驟 MEMO.AI 完全遺漏，由使用者實機補充）

步驟 5：第 1 份：描邊改為 0.25pt
         效果 → 扭曲與變形 → 鋸齒化
         大小：70%（相對的），鋸齒數：0
原文：选中第一个文字 打开效果 扭曲和变换 粗糙化 大小改为70 细节改为0

步驟 6：第 2 份：描邊改為 0.5pt
         鋸齒化，大小：6%（相對的），鋸齒數：1
原文：另外两份改为0.5 / 第二个文字 大小为6 细节为1

步驟 7：第 3 份：描邊改為 0.5pt
         鋸齒化，大小：2%（相對的），鋸齒數：1
原文：第三个文字 大小为2 细节为1

步驟 8：⏸ 確認效果
         詢問使用者目前效果是否滿意，可手動調整各份的鋸齒化數值

         【效果不符合時 → 字型重試流程】
         如果效果不明顯或完全看不出凌亂線條感：
           → 告知：目前字型可能不相容，建議換字型重試
           → 請使用者回到步驟 0，選擇不同字型
           → 從步驟 1 重新執行整個流程
           → 📝 記錄本次使用的字型名稱（無效字型）

         【效果滿意時】
           → 📝 記錄目前使用的字型名稱，新增至末節「已驗證字型」清單

步驟 9：確認後，全選三份文字 → 水平居中 + 垂直居中，完成疊合
原文：最后把他们三个剧中对齐
```

## 可調參數

| 參數名稱 | 預設值 | 說明 |
|---------|--------|------|
| 第 1 份描邊 | 0.25pt | 最細，主要輪廓線 |
| 第 2 份描邊 | 0.5pt | 中等粗細 |
| 第 3 份描邊 | 0.5pt | 同第 2 份 |
| 第 1 份鋸齒化大小 | 70%（相對的） | 最粗糙，凌亂感最強 |
| 第 2 份鋸齒化大小 | 6%（相對的） | 中等凌亂感 |
| 第 3 份鋸齒化大小 | 2%（相對的） | 輕微凌亂，保留字形輪廓 |
| 移動垂直距離 | 45mm | 步驟 4 的疊加間距 |
