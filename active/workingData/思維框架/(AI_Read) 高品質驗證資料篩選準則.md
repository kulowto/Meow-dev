# 高品質驗證資料篩選準則

## 核心問題

> **這個素材，真的適合作為論證依據嗎？**
> 還是只是萬中選一、學不來的倖存者偏差？

---

## 倖存者偏差（Survivorship Bias）

### 定義
只看到成功的案例，忽略了沉默的失敗案例，導致對某個方法或行為的效果過度高估。

### 辨識方法

| 特徵 | 說明 |
|------|------|
| 樣本極小 | 「有一個人這樣做成功了」 |
| 缺乏對照 | 沒有提及用同樣方法失敗的人有多少 |
| 結論跳躍 | 從個案直接推導出普遍規律 |
| 強調特殊 | 強調條件極端罕見（天才、極端環境等） |

### 範例
「A 大學輟學後創業成功，所以讀書沒用」——忽略了數十萬個輟學後失敗的人。

---

## 林迪效應（Lindy Effect）

### 定義
一件事物存在的時間越久，它未來能繼續存在的預期時間也越長。
時間本身是一種篩選機制——能被反覆驗證、長期留存的知識，往往才是真正可靠的。

### 應用到論證素材篩選

優先選用符合林迪效應的素材：

| 優先 | 說明 |
|------|------|
| ✅ 長時間考驗 | 數十年甚至數百年仍被廣泛引用的理論或研究 |
| ✅ 大樣本重複驗證 | 多個獨立研究得出相同結論 |
| ✅ 跨文化、跨時代有效 | 不只適用於特定情境 |
| ❌ 近期爆紅 | 尚未經過時間考驗 |
| ❌ 單一案例 | 特別是高知名度、條件極端的案例 |

---

## AI 評估框架

當使用者提供一個論點或數據作為依據時，AI 應評估：

1. **來源可信度**：是否有明確出處？出處是否具學術或長期認可？
2. **樣本充足性**：樣本多大？是否有對照組？
3. **倖存者偏差風險**：有沒有大量沉默的反例被忽略？
4. **林迪效應符合度**：這個論點存在多久了？有沒有被時間反覆驗證？
5. **環境遷移性**：這個結論在不同情境下是否仍然成立？

---

## 篩選結論分類

| 評估結果 | 建議處理方式 |
|---------|------------|
| 符合林迪效應、大樣本 | 可作為主要論證素材 |
| 近期研究但設計嚴謹 | 可輔助引用，需標注「尚待時間驗證」 |
| 倖存者偏差明顯 | 明確指出，不建議作為主要論據 |
| 個案故事 | 只能作為「說明例子」，不能作為「論證依據」 |
