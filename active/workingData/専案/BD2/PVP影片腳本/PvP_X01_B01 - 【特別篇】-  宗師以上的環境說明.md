# 【教學類-影片】

以前在【基礎篇】介紹過，宗師以上是另一個獨立運作的環境，概念會不同，那這邊我們就開一期給大家了解一下。

由於大多人根本沒想打到這段位，對於這種影片跟資訊完全沒需求，那不妨當個故事聽聽就好，也可以了解後面的環境是怎樣的。

- 什麼構成了玩家在哪個階層呢？

    當初有提到，宗師以下分三個主軸來評分，分別是「裝備」、「服裝練度」、「陣容主題性」。

    總排名前100的人，至少都具備了這些條件。

    但宗師3跟宗師2以上還是有個檻，主要差異在「提早多久打」跟「陣容的對策與細節」。

    而宗師2以上到衛冕者的環境，就是這些「個人實力的統籌」外加「團戰」的範疇了。

    想必有人會想說，「團戰」？ 鏡中不是單人PVP嗎？

    當初的我其實也有同樣的疑惑，千夜佬曾分享過一段故事，跟大家分享一下，後面就能理解為何環境不同。

- 棕二史記：鏡中之戰的變化

    雖然棕二是個很適合躺躺的遊戲，但其實很多人想拿鏡中第一。

    有的是認真玩PVP的人，有的是想要榮耀，而更多的是想拚「神臨」稱號的。

    想必大家都知道，「神臨」這是目前這遊戲的最高榮譽，需要魔獸、鏡中都拿過第一，才有實現可能，雖然後續有推出「領主」這個被歸類為同等級的稱號，但獲取難度大不相同，也因此只要你有「神臨」，帳號如果有打算轉賣，甚至價格翻幾倍的都有，不少人包含某些號商，其實是為了這個才想打第一的。

- PVP的機制：導致成「團戰」

    BD2的PVP有一個規則，就是越往上，匹配到的人範圍就越窄，比如第十名可以匹配到第5-20名，第一名只能匹配到第1-10名，這個規則，開服那陣子很早就有人發現了，同時那時候開始，BD2的PVP頂上戰爭，從個人戰轉為小型團戰，而後也有演變成「國戰」的狀況過。

- 回到故事起點

    最開始，運用這個規則的，是一名日本和兩名韓國玩家。
    當時他們3個人靠這個規則在PVP裡組隊，成為了BD2裡第一批時空超越者的玩家
    <###「時空超越者」的補充：「拿三次，鏡中之戰第一」才有的稱號 ###>

    那時榜上能說得上嘴的中文玩家不多，最有名的就「千夜佬」跟「暗殺佬」這些老登，自然也沒法參與這場「團戰」。

- 第35賽季，「團戰」時代的開始

    中文這邊的玩家也開始運用組隊的規則，開始對日韓玩家進行反擊，
    當時千夜佬、九澄佬、暗殺佬，組成三人小隊，和日韓那邊的玩家進行3v3對抗。

- 補充說明，爭排名技巧：卡人

    以千夜佬當時舉例，日本玩家たいふ一ん，他現在是第一名，而暗殺佬是第二名，我為此要把たいふ一ん的分數拉下去，因為前十名的範圍很窄，所以他必定能匹配到我。

    所以只要我知道了他的進攻，我就可以改變自己的防守去針對他的進攻，讓他匹配到我的時候必輸

    這也是為什麼很多人都強調爭PVP的排名，「變陣」和「拆陣」的反應速度很重要

- 第41-50賽季，「團戰」演變成「國戰」

    3人小隊和日韓玩家3人，3v3越來越激烈的時候，也有越來越多的玩家也發現了這個規則，此時歐美玩家也加入了戰場

    隨後，BD2的PVP就進入了一個三足鼎立的時代，中文社區、日韓社區、歐美社區，PVP就演變成「國戰」

- 第51-80賽季：「國戰」的形式漸漸退去

    但是再往後，因為日韓玩家的數量本來就不多
    歐美社區的玩家又因為時差，導致難以參與鏡中之戰的競爭

    同時，越來越多的中文玩家加入了鏡中之戰的競爭，比如九澄、葉心、葛蘭、默示...等等

    這導致歐美和日韓的組隊玩家，逐漸被中文社區的玩家所壓制

    這段時間歐美玩家和日韓玩家因為人數問題，越來越力不從心，所以當時老一輩的歐美和日韓玩家也逐漸退出了鏡中之戰的戰場

    之後，BD2的鏡中之戰基本都是被中文玩家統治了

- 第81賽季之後：轉回「團戰」，並在自己的團裡競爭

    近期仍有些其它國家的玩家也會去競爭PVP第一，像是「橘子」、「紅蛙」這些玩家，他們是真正意義上單槍匹馬，去鏡中之戰角逐第一的玩家。

    但如今BD2的鏡中之戰環境已經比較固化了，而非最初的個人戰，他們想爭第一，確實會遭到很多中文玩家的圍攻，也常常因為這樣不爽這些抱團的玩家，不過即便如此也並非不可能，之前有一期「紅蛙」這名玩家確實就是單槍匹馬，最後拿到了第一，跌破眾人眼鏡。

    不過其實中文玩家圈里，對PVP第一也是有競爭的，並不是說安排好誰拿誰就拿，如果是同組的，大多是先相互競爭，然後到10點左右看誰的分高，就停止競爭的；但如果遭遇到不同組的，就會一路競爭到最後。

這個故事就到這邊，想必大家可以理解，為何宗師以上的環境大不相同了吧。

我們回到正題，那「進攻/防守」的差異在哪呢？

- 「進攻」的核心概念？

    宗師以上，比的是「對策」、「換陣速度」。
    
    進攻時有些陣容，你要針對對手防守陣容的弱點，趁他病要他命，這些弱點通常是前面有說到的一些細節，並依此去改變裝備屬性、技能施放順序、站位...等方法來應對。
    
    若同時你的防守，能同時守到兩個人，分數就會上去。

- 「防守」的核心概念？

    宗師以上，「防守」是針對部分的玩家，此時防守的勝率大多不會超過18%。
    只要你能針對到一個人，分數便會維持住，針對到多個，便有機會再上分。
    
- 跟一般玩家的差異：「換陣」、「變陣」要快

    上面環境，「換陣」是個風險很高，卻有時不得不做的事。
    你「換陣」的那段時間，你會超光速掉分，因為周圍的人都在打你，而防守勝率在宗師以上基本只有不到兩成，但不換有可能會遇到分數卡住，上不去的問題，快速調整陣容是一個需要的技能。

    平時你會看到有些佬們，即便不在結算打很高分，其實就是在測陣容、練換陣，做這些準備，等到結算上戰場，就必須馬上應對，這塊也是目前我個人還要學的。

- 跟一般玩家的差異：結算時間多久前就開始打

    一般玩家，可能最多就一小時前打打排名卡個位。

    這群僅靠硬實力就能宗師2以上的玩家，為了角逐第一，六點開始打的案例也有，雖然最近大家幾乎都八點後才開始打，但每次要打就要花600酒的消耗，還不一定能上位成功，想必這是一般玩家無法想像的，而對於這些玩家是家常便飯。

# 收尾

這些事項跟上面那段故事聽完，想必也能讓不了解宗師以上環境的人，略知一二。

而從故事裡，也許各位或多或少能理解，為何前面我個人都偏好推薦千夜佬的影片，撇除蘑菇攻略組的因素，主要是在PVP這塊，千夜佬本人就是經歷很多場真槍實戰的戰場老兵，而他製作的這種影片含金量也是相當高的，哪怕現在環境改變了，但陣容的概念都還是能用的。

那這邊老話一句，如果覺得這期影片不錯的，各位不妨動動小手三連一下，你們做的這些小行為對於我們是莫大的幫助，之後我們還會盡可能地幫大家製作這些PVP影片，這邊先謝各位了

# 額外想法
## 蘑菇方：

### 影片長度的需求與建議
影片上，考量到逼哩逼哩的平台受眾，若可以再2~4分鐘內會最好。
油管上要有收益，通常要10分鐘以上，而這種能做成三個影片的整合版，一次上傳。

### 核心考量
有流量需求，但要讓聽眾願意幫我們做一件事，除了提醒之外，更重要的是有符合他們的需求，才有意願幫忙。

### 導流方法
如果影片之間有連結，可以做導流
譬如：這個陣容，對到「水人隊」是劣勢

跳個水人隊的影片連結；然後配音裡，可以說「若有忘記想複習或不知道什麼是水人隊的朋友，可以參考影片附上的連結」

備註：但也不能太頻繁使用，不然會變成觀眾反感
